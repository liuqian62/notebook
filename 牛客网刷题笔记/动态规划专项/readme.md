# 动态规划专项

这个目录用于集中整理动态规划相关题型，适合已经掌握基础题后专项训练。

## 这个目录适合什么时候用

- 已经理解基础动态规划概念，准备做专题强化
- 想把常见 DP 类型按“状态设计方式”重新梳理一遍
- 面试前需要快速回看不同 DP 题型的区别

## 推荐阅读顺序

1. [线性 dp](./线性dp.md)
2. [前缀和](./前缀和.md)
3. [差分](./差分.md)
4. [二维 dp](./二维dp.md)
5. [背包](./背包.md)
6. [区间 dp](./区间dp.md)
7. [树形 dp](./树形dp.md)
8. [状压 dp](./状压dp.md)
9. [数位 dp](./数位dp.md)

## 使用说明

- 如果还没系统做过动态规划，建议先从上层目录里的 [算法入门/动态规划](../算法入门/动态规划.md) 开始
- 这里更适合作为专项整理和复盘入口

## 建议复盘维度

- 这类题通常如何定义状态
- 转移方程依赖前哪些状态
- 初始化和边界条件最容易错在哪里
- 它和相邻题型的区别是什么
