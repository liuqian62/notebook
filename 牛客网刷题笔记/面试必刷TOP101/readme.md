# 面试必刷TOP101

这个目录按题型整理牛客常见高频题，适合面试前按专题快速复习。

## 这个目录适合什么时候用

- 已经做过一轮基础题，想按高频题型查缺补漏
- 面试前一到两周，希望按题型做集中复习
- 想快速找到“某一类题通常怎么做”

## 推荐刷题顺序

1. 链表
2. 二叉树
3. 堆 / 栈 / 队列
4. 哈希
5. 双指针
6. 字符串
7. 二分查找 / 排序
8. 递归 / 回溯
9. 动态规划
10. 贪心算法
11. 模拟

## 使用说明

- 适合按专题查缺补漏
- 如果只是准备笔试，可以配合上层目录里的 [高频榜单](../高频榜单.md) 一起看
- 如果还没系统刷过基础题，建议先看 [../算法入门/readme.md](../算法入门/readme.md)

## 复习建议

- 每个专题先挑 3 到 5 道代表题复盘思路
- 复习时重点记“常见模板 + 易错点 + 什么时候想到这种方法”
- 遇到明显薄弱项，再回到对应正文补题
