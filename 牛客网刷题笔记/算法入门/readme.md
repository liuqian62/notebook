# 算法入门

这个目录更适合刚开始系统刷题时使用，内容比 `面试必刷TOP101` 更偏基础和过渡。

## 这个目录解决什么问题

- 建立常见数据结构和基础算法的最小知识框架
- 在进入 `TOP101` 和 `动态规划专项` 前先补齐方法感
- 适合从“知道题型名”过渡到“知道该用什么方法”

## 推荐阅读顺序

1. [数据结构](./数据结构.md)
2. [搜索](./搜索.md)
3. [基础算法](./基础算法.md)
4. [动态规划](./动态规划.md)

## 目录说明

- `数据结构`：建立基础概念
- `搜索`：DFS / BFS 一类基础搜索方法
- `基础算法`：枚举、递归、分治等常见题型
- `动态规划`：作为进一步进入 DP 专项前的过渡

## 学完后往哪里走

- 想开始刷面试高频题：看 [../面试必刷TOP101/readme.md](../面试必刷TOP101/readme.md)
- 想专项补动态规划：看 [../动态规划专项/readme.md](../动态规划专项/readme.md)
