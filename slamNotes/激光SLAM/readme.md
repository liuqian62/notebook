# 激光SLAM

这个目录用于集中放激光 SLAM、激光惯导融合和相关扩展方向的资料入口。

当前本地笔记还不多，主要以方案索引和两篇学习笔记为主。

## 本地笔记

- [LIO-SAM 学习笔记](./LIO-SAM学习笔记.md)
- [LVI-SAM 学习笔记](./LVI-SAM学习笔记.md)

## 推荐阅读顺序

1. 先看 `LIO-SAM`
2. 再看 `LVI-SAM`
3. 之后按需要扩展到 FAST-LIO、Point-LIO、Cartographer、LOAM 等方案
