## slam 后端一般分为两种处理方法
* 扩展卡尔曼滤波（滤波方法）
* 图优化（非线性优化方法）

## 图优化
1. 构建图。机器人位姿作为顶点，位姿间关系作为边。
2. 优化图。调整机器人的位姿（顶点）来尽量满足边的约束，使得误差最小。
