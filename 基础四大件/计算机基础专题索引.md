# 计算机基础专题索引

这份文档承接目录导航之外的“速查入口”和“补充资料”，更适合面试前做最后一轮查漏补缺。

它和 [readme.md](./readme.md) 的分工是：

- `readme.md`：总入口、阅读顺序、目录导航
- `计算机基础专题索引.md`：按面试场景快速回看知识点

## 一页速查建议

- 想补数据结构和算法：先看 [数据结构](./数据结构.md) 和 [算法](./算法.md)
- 想补系统方向：先看 [OS](./OS.md) 和 [计组](./计组.md)
- 想补网络方向：先看 [计网](./计网.md)
- 想补后端常见问法：看 [mysql](./mysql.md) 和 [设计模式](./设计模式.md)

## 按场景速查

- 想准备概念对比题：优先看“数据结构 / 算法 / OS / 计网”里的对比小节
- 想准备原理题：优先看 [计组](./计组.md)、[OS](./OS.md)、[mysql](./mysql.md)
- 想准备后端面：优先看 [mysql](./mysql.md)、[设计模式](./设计模式.md)、[计网](./计网.md)
- 想做一轮面试前总复习：先回到 [readme.md](./readme.md) 按推荐顺序过一遍，再用这页查漏补缺
